# soon — 계정·데이터 삭제 방법

soon 앱 사용자는 본인의 계정과 모든 데이터를 다음 두 가지 방법으로 삭제할 수 있습니다.

---

## 방법 1. 앱 내에서 직접 삭제 (권장)

가장 빠르고 즉시 처리됩니다.

1. soon 앱 실행 후 본인 계정으로 로그인
2. 하단 탭 **더보기** → **프로필** 진입
3. 화면 하단의 **"회원 탈퇴"** 링크 클릭
4. 안내 문구 확인 후 화면에 표시된 **6자리 영문·숫자 확인 코드**를 입력
5. **"탈퇴하기"** 버튼 탭

→ **즉시** 본인의 모든 앱 데이터가 영구 삭제되고 자동 로그아웃됩니다.

---

## 방법 2. 이메일 요청

앱에 접근할 수 없는 경우, 아래 이메일로 요청해주세요.

**이메일**: [daehyun.kim@kccc.org](mailto:daehyun.kim@kccc.org)

요청 시 다음 정보를 포함해주세요:
- 본인 CCC 계정 ID 또는 가입 시 등록한 이메일
- 삭제 요청 사유 (선택)

→ 본인 확인 후 **영업일 기준 7일 이내** 처리됩니다.

---

## 무엇이 삭제되나요?

회원 탈퇴 시 **다음 데이터가 모두 영구 삭제**됩니다:

- 계정 정보 (이름, 이메일, 전화번호, 캠퍼스 정보 등)
- QT 묵상 기록
- 기도제목 및 기도 참여 기록
- 피드 게시물, 댓글, 좋아요
- 순(소그룹) 멤버십, 모임 기록
- 출석 기록
- 친구 소개 / 가입 신청
- 일정, 알림 이력
- 챌린지 인증 사진 및 응원 이력
- 푸시 알림 토큰
- 인증 정보 (Auth)

업로드된 사진은 시스템에서 함께 삭제 또는 익명화 처리됩니다.

---

## 삭제되지 않는 데이터

- **CCC 계정 자체**: 본 앱은 CCC 계정 기반으로 운영되며, CCC 회원 탈퇴는 [kccc.org](https://kccc.org)에서 별도로 진행됩니다.
- **법령에 따른 보관 의무 정보**: 관계 법령에서 일정 기간 보관을 요구하는 경우 해당 기간 보관 후 파기됩니다.

---

## 처리 기간

- **앱 내 탈퇴**: 즉시 (트랜잭션 완료 후 영구 삭제)
- **이메일 요청**: 영업일 기준 **7일 이내**
