# 라운지스 오피스(LOUNGEES Office) 개인정보처리방침

**시행일**: 2026년 7월 3일

라운지스 오피스는 **운영자 본인(김대현)만 사용하는 비공개 내부 업무 자동화 도구**입니다. 일반 사용자를 대상으로 서비스를 제공하지 않으며, 제3자의 개인정보를 수집·저장·공유하지 않습니다.

---

## 1. 수집하는 정보

운영자 본인의 Google 계정(Gmail, Google Calendar) 데이터에만 접근합니다.

| 목적 | 접근 데이터 |
|---|---|
| 메일 분류·요약, 조치 필요 여부 판단 | Gmail 메일 제목·발신자·본문 일부(메타데이터), 필요 시 답장 발송 |
| 일정 조회·등록·수정 | Google Calendar 일정 |

수집된 데이터는 운영자가 관리하는 비공개 데이터베이스(Supabase)에만 저장되며, 제3자에게 제공되지 않습니다.

---

## 2. Google API 사용 범위

본 애플리케이션이 요청하는 Google API 스코프는 다음과 같습니다.

- `gmail.readonly` — 메일 조회
- `gmail.send` — 메일 발송
- `calendar` — 캘린더 조회·수정

본 애플리케이션의 Google API 사용은 [Google API Services User Data Policy](https://developers.google.com/terms/api-services-user-data-policy)의 제한된 사용(Limited Use) 요건을 준수합니다. 수집한 Google 사용자 데이터는 위에 명시된 목적 외에 사용하지 않으며, 광고 목적으로 이용하거나 인간에게 열람시키지 않습니다.

---

## 3. 보유 및 파기

운영자 본인 계정 데이터는 서비스 이용 기간 동안 보유하며, 연동 해제 시 지체 없이 파기합니다.
