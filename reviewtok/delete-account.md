# 리뷰톡 — 계정·데이터 삭제 방법

리뷰톡 사용자는 본인의 계정과 모든 데이터를 다음 두 가지 방법으로 삭제할 수 있습니다.

---

## 방법 1. 앱 내에서 직접 삭제 (권장)

가장 빠르고 즉시 처리됩니다.

1. 리뷰톡 앱(iOS·Android)을 실행해 **카카오 또는 애플로 로그인** — 또는 토스 앱에서 리뷰톡 미니앱 실행 후 토스 계정으로 로그인
2. 하단 탭 **마이** 진입
3. **프로필** → **회원 탈퇴**
4. 안내 문구 확인 후 화면에 표시된 **6자리 영문·숫자 확인 코드**를 입력
5. **"탈퇴하기"** 버튼 탭

→ **즉시** 본인의 모든 데이터가 영구 삭제되고 자동 로그아웃됩니다.

### 로그인 연결 해제 (선택)

회원 탈퇴와 별개로 로그인 제공자 측에서도 연결을 해제할 수 있습니다.

- **토스**: 토스 앱 → 마이 → 연결한 서비스 관리 → 리뷰톡 연결 해제
- **카카오**: 카카오 계정 설정 → 연결된 서비스에서 해제
- **애플**: iOS 설정 → Apple ID → 로그인 항목 → 리뷰톡 사용 중단

연결 해제만으로는 리뷰톡에 보관된 회원 데이터가 즉시 삭제되지 않으며, 보유 기간 후 파기됩니다. 즉시 삭제를 원하면 위 회원 탈퇴 절차를 진행해 주세요.

## 무엇이 삭제되나요?

회원 탈퇴 시 **다음 데이터가 모두 영구 삭제**됩니다:

- 계정 정보 (이름, 휴대전화번호, 토스 CI 또는 카카오·애플 계정 식별자, 이메일 등)
- 리뷰 상품 데이터 (상품명, 상품가, 리뷰비, 상태 정보)
- 판매자 정보 (판매자명, 입금자명, 채팅 링크)
- 결제수단 정보 (카드 별칭, 카드사명, 카드 끝 4자리)
- 계정 별칭 등 부가 정보
- 디바이스 인증 토큰
- 서비스 이용 기록

## 연동 서비스와 연결된 경우

라운지스2 또는 새 데스크톱 브랜드와 연동된 계정의 경우:

- **리뷰톡에서만 탈퇴**: 리뷰톡 측 인증·디바이스 토큰은 삭제되지만, 연동 서비스의 계정과 데이터는 그대로 유지됩니다.
- **연동 서비스에서도 탈퇴 필요**: 모든 데이터를 완전히 삭제하려면 각 서비스에서도 탈퇴를 진행해 주세요.

데이터가 양쪽에서 통합 조회되는 구조이므로, 정확한 삭제 범위는 이메일로 문의해 주시면 안내해 드립니다.
